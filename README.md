Japanese chess implementation made with ReactJS.

So far implemented:

* Pieces movement
* Promoted pieces
* Movement of promoted pieces

To implement:
* Re-introduce captured pieces
* Check and checkmate

#### Installation and run(local)

1. Clone or download
2. `npm install`
3. `npm run start`
4. On browser go to: [localhost:3000](http://localhost:3000)
